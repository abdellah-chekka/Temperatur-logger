## Temperaturlogger - Technisches Monitoring-System

Entwickelt von Abdellah Chekka als Aufgabe an der Universität hochgeladen für meine Bewerbung für die Ausbildungsstelle als Fachinformatiker für Anwendungsentwicklung.

## Über das Projekt

Ein C++ Programm zur Überwachung und Analyse von Temperaturdaten in technischen Umgebungen.

## Funktionen

- Temperaturüberwachung mit Grenzwerten
- Automatische Warnungen bei Überschreitung
- Statistische Analyse (Min, Max, Durchschnitt)
- Speicherung der Messdaten

## Kompilieren und Ausführen

```bash
g++ temperaturlogger.cpp -o logger
./logger
